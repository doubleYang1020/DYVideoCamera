## Installation

DYVideoListKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'DYVideoListKit'
```
## Caution
your podfile should comment out use_frameworks!
```ruby
#use_frameworks!
```
